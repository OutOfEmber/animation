1.Как выполнить поворот, искажение, перемещение и масштабирование объекта?

Для этого используется свойство transform с соответствующими функциями:
Поворот: rotate(угол)
Пример: transform: rotate(45deg); (поворот на 45 градусов по часовой стрелке)
Искажение (Скос): skew(X-угол, Y-угол)
Пример: transform: skew(10deg, 5deg); (скос по оси X на 10deg, по оси Y на 5deg)
Перемещение: translate(X-смещение, Y-смещение)
Пример: transform: translate(100px, 50px); (сдвиг на 100px вправо и 50px вниз)
Масштабирование: scale(X-масштаб, Y-масштаб)
Пример: transform: scale(1.5, 0.8); (увеличить ширину в 1.5 раза, уменьшить высоту в 0.8 раза)

2.Что такое точка трансформации объекта? Как изменить точку трансформации?

Точка трансформации — это точка, вокруг которой происходят преобразования (например, вращение или масштабирование). По умолчанию она находится в центре элемента (50% 50%).
Изменить точку трансформации можно с помощью свойства transform-origin.
Синтаксис: transform-origin: x-pos y-pos;
Примеры:
transform-origin: top left; (точка в левом верхнем углу)
transform-origin: 0 0; (то же самое, что и top left)
transform-origin: 20px 80%; (20px от левого края и 80% от верхнего)
transform-origin: center; (по центру, значение по умолчанию)
