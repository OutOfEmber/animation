1.Как выполнить поворот, искажение, перемещение и масштабирование объекта?

Для этого используется свойство transform с соответствующими функциями:
Поворот: rotate(угол)
Пример: transform: rotate(45deg); (поворот на 45 градусов по часовой стрелке)
Искажение (Скос): skew(X-угол, Y-угол)
Пример: transform: skew(10deg, 5deg); (скос по оси X на 10deg, по оси Y на 5deg)
Перемещение: translate(X-смещение, Y-смещение)
Пример: transform: translate(100px, 50px); (сдвиг на 100px вправо и 50px вниз)
Масштабирование: scale(X-масштаб, Y-масштаб)
Пример: transform: scale(1.5, 0.8); (увеличить ширину в 1.5 раза, уменьшить высоту в 0.8 раза)

2.Что такое точка трансформации объекта? Как изменить точку трансформации?

Точка трансформации — это точка, вокруг которой происходят преобразования (например, вращение или масштабирование). По умолчанию она находится в центре элемента (50% 50%).
Изменить точку трансформации можно с помощью свойства transform-origin.
Синтаксис: transform-origin: x-pos y-pos;
Примеры:
transform-origin: top left; (точка в левом верхнем углу)
transform-origin: 0 0; (то же самое, что и top left)
transform-origin: 20px 80%; (20px от левого края и 80% от верхнего)
transform-origin: center; (по центру, значение по умолчанию)

3.Как применить несколько трансформаций к одному объекту? Приведите примеры.
Несколько трансформаций применяются к одному объекту путем перечисления функций в свойстве transform через пробел. Важно: порядок функций имеет значение, так как каждая последующая трансформация применяется после предыдущей.
Пример 1 (Сначала сдвиг, потом поворот):
.element {
  transform: translateX(100px) rotate(45deg);
}
Пример 2 (Сначала поворот, потом сдвиг):
.element {
  transform: rotate(45deg) translateX(100px);
}
Во втором примере элемент сначала повернется, а затем будет сдвинут уже вдоль новой, повернутой оси X.

4.Как применить несколько эффектов перехода к одному объекту? Приведите примеры.

Для применения нескольких переходов к разным свойствам их нужно перечислить через запятую в свойстве transition.
Синтаксис: transition: свойство1 время1 функция1 задержка1, свойство2 время2 функция2 задержка2, ...;

Пример:

.button {
  background-color: blue;
  width: 100px;
  transition: background-color 0.3s ease, width 0.5s linear 0.2s;
}
.button:hover {
  background-color: red;
  width: 150px;
}
В этом примере при наведении:
Цвет фона изменится за 0.3 секунды с функцией ease.
Ширина изменится за 0.5 секунды с функцией linear, но начнет меняться только через 0.2 секунды (задержка).

5.Какие временные функции для анимации вы знаете? Как задать пользовательскую функцию?

Временная функция (animation-timing-function / transition-timing-function) определяет скорость выполнения анимации в течение ее продолжительности.
Стандартные функции:
ease (по умолчанию) - медленный старт, затем быстро, медленное завершение.
linear - постоянная скорость.
ease-in - медленный старт.
ease-out - медленное завершение.
ease-in-out - медленный старт и медленное завершение.
step-start - мгновенно переходит в конечное состояние.
step-end - остается в начальном состоянии до конца анимации, затем мгновенно переходит в конечное.
steps(int, start|end) - разбивает анимацию на указанное количество шагов.
Пользовательская функция (Кривая Безье):
Задается с помощью функции cubic-bezier(x1, y1, x2, y2), где x1, y1, x2, y2 - координаты двух контрольных точек, определяющих форму кривой.
Пример: animation-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55); (создает "пружинный" эффект)

6.Как задать ход выполнения анимации?

Ход выполнения анимации задается с помощью правила @keyframes. Внутри этого правила определяются ключевые кадры (стили) в определенные моменты времени.
Синтаксис:
@keyframes имя-анимации {
  from {
    /* Стили в начале анимации */
  }
  to {
    /* Стили в конце анимации */
  }
  /* ИЛИ с использованием процентов */
  0% {
    /* Стили в 0% времени анимации */
  }
  50% {
    /* Стили в 50% времени анимации */
  }
  100% {
    /* Стили в 100% времени анимации */
  }
}
Пример:
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
.box {
  animation: slide-in 1s ease-in-out;
}

7.Какие CSS3 свойства анимации вы знаете?

Основные свойства для управления анимацией:
animation-name - имя анимации (из @keyframes).
animation-duration - продолжительность анимации (например, 2s).
animation-timing-function - временная функция.
animation-delay - задержка перед началом анимации.
animation-iteration-count - количество повторений (infinite для бесконечного).
animation-direction - направление (normal, reverse, alternate, alternate-reverse).
animation-fill-mode - определяет, какие стили применяются до начала и после завершения анимации (none, forwards, backwards, both).
animation-play-state - состояние анимации (running или paused).

8.Опишите формат краткой записи анимации.

Свойство animation позволяет объединить все отдельные свойства анимации в одной строке.
Синтаксис:
animation: name duration timing-function delay iteration-count direction fill-mode play-state;
Порядок значений важен! Обязательными являются только animation-name и animation-duration.
Пример полной записи:
.element {
  animation-name: my-animation;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}
Эквивалентная краткая запись:
.element {
  animation: my-animation 2s ease-in-out 1s infinite alternate;
}
Минимальная запись (только имя и длительность):
.element {
  animation: my-animation 2s;
}
